## üêõ TROUBLESHOOTING

### Issue: "API key not found"
**Solution**: 
- Add `VITE_GEMINI_API_KEY` to `.env.local`
- Or set in localStorage: `localStorage.setItem('gemini_api_key', 'your-key')`

### Issue: "Camera not working"
**Solution**:
- Check browser permissions
- Try HTTPS (some browsers require it)
- Check browser console for errors

### Issue: "Voice not recognized"
**Solution**:
- Check microphone permissions
- Ensure browser supports Web Speech API
- Test with simple words first

### Issue: "Gemini not responding"
**Solution**:
- Verify API key is valid
- Check rate limits (free tier has limits)
- Check internet connection
- Review browser console for errors

### Issue: "Learned items not persisting"
**Solution**:
- Check localStorage is enabled
- Check browser storage is not full
- Clear and retry
