## üîß TROUBLESHOOTING

### API Key Not Working?
- Check key is valid at https://aistudio.google.com/app/apikey
- Verify it's in `.env.local` or localStorage
- Restart dev server after adding key

### Camera Not Opening?
- Check browser permissions
- Try HTTPS (required by some browsers)
- Use Firefox/Chrome (best support)

### Voice Not Recognized?
- Check microphone permissions
- Ensure browser supports Web Speech API
- Test with simple words first
- Check speaker volume

### Getting "Gemini not responding"?
- Check internet connection
- Verify API key is valid
- Wait - initial response may take a few seconds
